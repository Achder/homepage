# Splats

The `/splat` page renders a SuperSplat / Gaussian Splatting scene from this folder.

Drop your exported file here as:

```
public/splats/scene.sog
```

Spark also accepts `.ply`, `.spz`, `.splat`, `.ksplat`, and `.zip` — to use a
different file, change `SPLAT_URL` in `src/pages/splat.astro`.
